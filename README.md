# نماء — Namaa Live Simulation

نموذج أولي تفاعلي لمنصة عربية تحول أفكار الكتب إلى حلقات صوتية ثرية ومسارات تعلم عملية، مع تصور مستقبلي لإنتاجات `Namaa Originals` بالتعاون مع دور النشر.

## ما يعمل في النسخة الحالية

- واجهة عربية RTL متجاوبة.
- 17 حلقة تجريبية و7 تصنيفات و3 مسارات تعلم.
- بحث محلي بالعناوين والمؤلفين والموضوعات.
- صفحات تفصيلية للحلقات.
- مشغل مرئي يحاكي التشغيل دون ملفات صوت.
- حفظ الحلقات محليًا داخل المتصفح.
- بوسترات وهوية بصرية مؤقتة قابلة للاستبدال.

## ما لا تتضمنه النسخة الحالية

- لا توجد ملفات صوت حقيقية.
- لا توجد قاعدة بيانات إنتاج.
- لا يوجد تسجيل دخول أو دفع حقيقي.
- لا يحتاج المشروع إلى Turso أو Neon أو أي متغيرات بيئية.

## التشغيل محليًا

```bash
npm install
npm run dev
```

ثم افتح `http://localhost:3000`.

## الاختبار والبناء

```bash
npm run lint
npm run build
```

## النشر على Vercel

1. ارفع الكود إلى مستودع GitHub.
2. اربط المستودع بحساب Vercel.
3. اضغط Deploy دون إضافة قاعدة بيانات أو Environment Variables.

## الملفات البصرية

- `public/brand/`: نسخ الشعار والأيقونة.
- `public/covers/`: أغلفة الحلقات التجريبية المولدة للمنصة.
- `public/posters/`: بوسترات العرض وNamaa Originals.

## التطوير المستقبلي

عند اعتماد المنتج يمكن استبدال بيانات `src/data/demo.ts` بطبقة API وقاعدة بيانات، ثم ربط التخزين الصوتي والحسابات والاشتراكات دون إعادة تصميم الواجهة.
